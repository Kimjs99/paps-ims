# PAPS-IMS 교사 사용 가이드

> **PAPS-IMS**는 Google Sheets를 데이터베이스로 사용하는 학생체력평가(PAPS) 관리 시스템입니다.
> 설치 프로그램이 없으며, 크롬 브라우저와 Google 계정만 있으면 사용할 수 있습니다.

---

## 1. 시작하기 전에

**필요한 것**
- Google 계정 (학교 Google Workspace 또는 개인 계정)
- Chrome 브라우저 (권장) — Safari, Edge도 지원
- 앱 URL: `https://paps-ims.vercel.app`

---

## 2. 최초 설정 (1회만)

### 2-1. Google Sheet 사본 만들기

1. 아래 링크에서 템플릿 Sheet를 엽니다
   > 템플릿 링크는 앱 온보딩 화면에서 확인
2. 상단 메뉴 **파일 → 사본 만들기** 클릭
3. 사본 이름 입력 후 **확인** (예: `2026 화접중학교 PAPS`)
4. 사본이 내 Google Drive에 저장됩니다

### 2-2. Sheet ID 확인하기

사본 Sheet URL에서 Sheet ID를 복사합니다.

```
https://docs.google.com/spreadsheets/d/【여기가 Sheet ID】/edit
```

예시:
```
https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit
                                        ↑ 이 부분 복사
```

### 2-3. 앱 최초 설정

1. 앱 URL 접속 → **Google 로그인** 버튼 클릭
2. Google 계정 선택 및 권한 허용
3. **Sheet ID 입력** → 복사한 ID 붙여넣기 → **연동 테스트** 클릭
4. "연동 성공" 메시지 확인
5. 학교명, 담당교사명, 학년도 입력 → **완료**

---

## 3. 학생 명단 등록

### CSV 파일로 일괄 등록 (권장)

**CSV 형식** (엑셀에서 "다른 이름으로 저장 → CSV" 로 저장):

```
student_id,name,gender,grade,class,height,weight
S001,홍길동,M,1,3,170,65
S002,김영희,F,1,3,162,52
```

- `gender`: 남자 `M`, 여자 `F`
- `grade`: 학년 숫자 (1~3)
- `class`: 반 숫자

**등록 방법**:
1. 좌측 메뉴 **학생** → **CSV 업로드** 버튼
2. 파일 선택 → 미리보기 확인 → **업로드**

### 개별 등록

1. **학생** 메뉴 → **+ 학생 추가** 버튼
2. 정보 입력 후 **저장**

---

## 4. 측정 데이터 입력

### 학급 단위 입력 (권장)

1. 홈 화면에서 학급 카드 선택 (예: 1학년 3반)
2. 심폐지구력 종목 선택 (왕복오래달리기 / 오래달리기-걷기 / 스텝검사)
3. 학생별 측정값 입력 → 등급이 자동 계산됩니다
4. **학급 전체 저장** 클릭

> 💡 저장 전에 자리를 비워야 할 경우 임시저장됩니다. 재접속 시 이어서 입력할 수 있습니다.

### 측정 항목

| 체력 요소 | 측정 종목 |
|----------|---------|
| 심폐지구력 | 왕복오래달리기, 오래달리기-걷기, 스텝검사 |
| 근력·근지구력 | 윗몸말아올리기, 악력 |
| 유연성 | 앉아윗몸앞으로굽히기 |
| 순발력 | 50m 달리기, 제자리멀리뛰기 |
| BMI | 키·몸무게 자동 계산 |

---

## 5. 대시보드 및 분석

1. 상단 메뉴 **대시보드** 클릭
2. **학년/반/성별** 필터로 원하는 집단 선택
3. KPI 카드, 등급 분포, 레이더 차트 확인

> 데이터는 30초마다 자동으로 새로고침됩니다.

---

## 6. 보고서 출력

1. 대시보드 → **보고서** 탭
2. **학급 보고서**: 학년·반 선택 → **PDF 내보내기**
3. **개인 성장 카드**: 학생별 체력 성장 카드 → 전체 PDF 일괄 출력 가능
4. **원시 데이터**: 전체 측정 데이터 엑셀(.xlsx) 내보내기

## 7. 자주 묻는 질문 (FAQ)

**Q. 로그인 후 데이터가 안 보입니다**
→ 새로고침(F5) 후 확인. 계속되면 **설정 → 데이터 새로고침** 버튼 클릭.

**Q. "연동 테스트 실패" 오류가 납니다**
→ Sheet ID를 다시 확인하세요. URL의 `/d/` 뒤에서 `/edit` 앞까지의 문자열만 입력해야 합니다.
→ Sheet가 본인 Google 계정으로 공유되어 있는지 확인하세요.

**Q. 인터넷이 끊겼을 때 입력 중이었습니다**
→ 측정 데이터는 임시저장됩니다. 인터넷 복구 후 재접속하면 이어서 저장할 수 있습니다.

**Q. 성적 기준(등급)이 달라 보입니다**
→ PAPS 공식 기준표(교육부)를 사용합니다. 업데이트가 필요하면 `grades_standard` 시트를 수정하세요.

**Q. 학생을 잘못 등록했습니다**
→ **학생** 메뉴에서 해당 학생 행을 클릭해 수정하거나 비활성화할 수 있습니다.

**Q. "Google Sheets 업데이트가 필요합니다" 배너가 뜹니다**
→ **Sheet 자동 업데이트** 버튼을 클릭하면 자동으로 처리됩니다.
