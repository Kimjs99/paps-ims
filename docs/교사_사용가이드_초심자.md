# PAPS-IMS 교사 사용 가이드 (초심자용)

> 이 가이드는 컴퓨터나 앱 사용이 익숙하지 않은 선생님을 위해 모든 단계를 사진 설명 수준으로 풀어썼습니다.
> 처음부터 끝까지 순서대로 따라오시면 누구든 사용할 수 있습니다.

---

## 이 앱이 뭔가요?

**PAPS-IMS**는 학생 체력검사(PAPS) 결과를 컴퓨터로 기록하고 분석하는 프로그램입니다.

- 설치할 것이 없습니다 — 인터넷 브라우저에서 바로 사용합니다
- 측정값을 입력하면 등급이 자동으로 계산됩니다
- 학급별·학생별 통계와 보고서를 자동으로 만들어 줍니다
- 데이터는 선생님 본인의 Google Drive(구글 드라이브)에 안전하게 저장됩니다

---

## 준비물 확인 (시작 전 반드시 확인)

시작하기 전에 아래 3가지가 있는지 확인하세요.

### 1. Google 계정

Google 계정은 Gmail 주소(`@gmail.com`)가 있으면 이미 있는 것입니다.
학교에서 제공한 Google Workspace 계정(`@학교도메인`)도 사용할 수 있습니다.

> **Google 계정이 없다면?**
> 1. [accounts.google.com](https://accounts.google.com) 접속
> 2. "계정 만들기" 클릭 후 안내에 따라 가입

---

### 2. 인터넷 브라우저

**Chrome(크롬) 브라우저**를 권장합니다. Safari(사파리), Edge(엣지)도 사용 가능합니다.

> **크롬이 없다면?**
> 1. 현재 브라우저에서 `google.com/chrome` 접속
> 2. "Chrome 다운로드" 클릭 후 설치

---

### 3. 앱 주소

앱 주소: **`https://paps-ims.vercel.app`**

브라우저 주소창에 위 주소를 입력하고 Enter를 누르면 됩니다.

---

## STEP 1. 처음 한 번만 하는 초기 설정

> 이 설정은 **처음 사용할 때 딱 한 번**만 하면 됩니다.
> 이후에는 바로 로그인해서 사용할 수 있습니다.

초기 설정은 총 5단계로 구성되어 있으며, 앱이 안내해 줍니다.

---

### 1단계 — Google 로그인

1. 브라우저에서 `https://paps-ims.vercel.app` 접속
2. 화면 중앙에 **"Google 계정으로 로그인"** 버튼이 보입니다 → 클릭
3. 새 창(팝업)이 열리면서 Google 계정 목록이 나타납니다
4. 사용할 계정을 클릭하세요
5. **"허용"** 버튼을 클릭하세요 (앱이 Google Drive와 Sheets에 접근하는 권한을 요청합니다)
6. 팝업 창이 자동으로 닫히고 앱 화면으로 돌아옵니다

> **팝업이 차단됐다는 메시지가 뜨면?**
> 브라우저 주소창 오른쪽 끝에 작은 아이콘이 생깁니다. 클릭 후 "팝업 허용"을 선택하고 다시 로그인 버튼을 누르세요.

---

### 2단계 — Google Sheets 사본 만들기

이 앱은 데이터를 **선생님 본인의 Google Sheets(구글 스프레드시트)**에 저장합니다.
먼저 앱 전용 시트를 내 드라이브에 만들어야 합니다.

1. 화면에 **"Sheets 사본 만들기"** 버튼이 있습니다 → 클릭
2. 새 탭(또는 새 창)에서 Google Sheets 템플릿이 열립니다
3. 화면 상단 메뉴에서 **파일** 클릭
4. 드롭다운 메뉴에서 **"사본 만들기"** 클릭

   > 메뉴가 안 보이면 화면 상단 왼쪽의 메뉴 막대를 찾아보세요.

5. "문서 복사" 창이 뜨면 이름을 입력합니다
   - 예시: `2026 화접중학교 PAPS` (학교명과 연도를 넣으면 나중에 찾기 쉽습니다)
6. **"사본 만들기"** 버튼 클릭
7. 내 Google Drive에 새 Sheets 파일이 생성됩니다

---

### 3단계 — Sheet ID 복사하기

방금 만든 Sheets 파일의 고유 주소(ID)를 앱에 알려줘야 합니다.

1. 방금 생성된 Sheets 파일이 열려 있는 탭으로 이동합니다
2. 브라우저 주소창을 클릭하면 주소 전체가 보입니다

   주소가 이렇게 생겼습니다:
   ```
   https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit
   ```

3. 주소에서 `/d/` 바로 다음부터 `/edit` 바로 앞까지의 문자열이 **Sheet ID**입니다

   ```
   https://docs.google.com/spreadsheets/d/ [여기부터] /edit
                                             ↓
                              1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms
   ```

4. 그 문자열을 드래그로 선택한 뒤 **Ctrl+C** (Mac: Command+C)로 복사합니다

> **팁**: 문자열 길이가 40~50자 정도 됩니다. 너무 짧거나 길면 잘못 선택한 것입니다.

---

### 4단계 — Sheet 연동 테스트

1. 앱 탭으로 돌아옵니다 (탭을 잘못 닫았으면 `https://paps-ims.vercel.app` 다시 접속)
2. **"Sheet ID 입력"** 칸에 클릭 후 **Ctrl+V** (Mac: Command+V)로 붙여넣기
3. **"연동 테스트"** 버튼 클릭
4. "연동 성공!" 메시지가 나오면 정상입니다

> **"연동 실패" 오류가 나면?**
> - Sheet ID를 다시 확인하세요. `/d/` 뒤부터 `/edit` 앞까지만 복사했는지 확인
> - 복사할 때 앞뒤로 빈칸(공백)이 포함됐을 수 있습니다 → 입력 칸의 내용을 지우고 다시 붙여넣기

---

### 5단계 — 학교 정보 입력

1. **학교명** 입력 (예: `화접중학교`)
2. **담당교사명** 입력 (예: `김종석`)
3. **학년도** 입력 (예: `2026`)
4. **"완료"** 버튼 클릭

초기 설정이 끝났습니다! 이제 앱 홈 화면으로 이동합니다.

---

## STEP 2. 학생 명단 등록

> 측정 데이터를 입력하려면 먼저 학생 명단이 있어야 합니다.

---

### 방법 A. 엑셀 파일로 일괄 등록 (학생이 많을 때 권장)

#### A-1. 엑셀 파일 준비

엑셀(Excel) 또는 한셀에서 아래 형식으로 파일을 만드세요.

| student_id | name | gender | grade | class | height | weight |
|-----------|------|--------|-------|-------|--------|--------|
| S001 | 홍길동 | M | 1 | 3 | 170 | 65 |
| S002 | 김영희 | F | 1 | 3 | 162 | 52 |
| S003 | 이민준 | M | 2 | 1 | 175 | 68 |

**각 항목 설명:**

| 항목 | 설명 | 예시 |
|------|------|------|
| `student_id` | 학번 (중복 불가) | S001, 20260101 |
| `name` | 학생 이름 | 홍길동 |
| `gender` | 성별: 남자는 `M`, 여자는 `F` | M 또는 F |
| `grade` | 학년 숫자 | 1, 2, 3 |
| `class` | 반 숫자 | 1, 2, 3 ... |
| `height` | 키 (cm 단위, 숫자만) | 170 |
| `weight` | 몸무게 (kg 단위, 숫자만) | 65 |

#### A-2. CSV 파일로 저장

엑셀 파일을 **CSV 형식**으로 저장해야 합니다.

1. 엑셀에서 **파일** → **다른 이름으로 저장**
2. 파일 형식을 **"CSV (쉼표로 분리) (*.csv)"** 선택
3. 저장

> **한셀(한컴오피스)을 사용한다면?**
> **파일** → **다른 이름으로 저장** → 파일 형식 **"CSV"** 선택

#### A-3. 앱에 업로드

1. 앱 왼쪽 메뉴에서 **"학생"** 클릭
2. 오른쪽 상단의 **"CSV 업로드"** 버튼 클릭
3. 방금 저장한 CSV 파일 선택
4. 미리보기 화면에서 데이터가 올바른지 확인
5. **"업로드"** 버튼 클릭

업로드가 완료되면 학생 목록이 화면에 표시됩니다.

---

### 방법 B. 한 명씩 직접 입력

학생 수가 적거나 일부 학생만 추가할 때 사용합니다.

1. 앱 왼쪽 메뉴에서 **"학생"** 클릭
2. 오른쪽 상단 **"+ 학생 추가"** 버튼 클릭
3. 학번, 이름, 성별, 학년, 반, 키, 몸무게 입력
4. **"저장"** 버튼 클릭

한 명씩 반복해서 등록합니다.

---

### 학생 정보 수정하기

등록한 학생 정보가 잘못됐을 때:

1. **"학생"** 메뉴에서 해당 학생 행(줄)을 클릭
2. 수정 화면이 열리면 내용 변경 후 **"저장"**

---

### 학생 비활성화 (전학 등)

전학을 가거나 더 이상 측정이 필요 없는 학생은 삭제하지 말고 **비활성화**하세요.
(비활성화하면 통계에서 제외되지만 데이터는 보존됩니다.)

1. 해당 학생 클릭
2. **"비활성화"** 또는 활성여부 토글 끄기
3. **"저장"**

---

## STEP 3. 체력 측정 데이터 입력

> 체육 시간에 측정한 결과를 입력하는 단계입니다.

---

### 학급 단위로 입력하기 (가장 많이 사용하는 방법)

1. 앱 홈 화면에서 **학급 카드**를 클릭합니다
   - 예: "1학년 3반" 카드 클릭
2. 측정 화면이 열리면 **심폐지구력 종목**을 먼저 선택합니다
   - 왕복오래달리기
   - 오래달리기-걷기
   - 스텝검사
   (셋 중 학교에서 사용하는 종목 하나 선택)
3. 학생 목록이 나타납니다. 각 학생 옆에 **측정값을 입력**합니다
   - 숫자 입력 칸을 클릭 후 값 입력
   - 입력하면 오른쪽에 등급이 자동으로 표시됩니다
4. 모든 학생의 값을 입력했으면 화면 하단의 **"학급 전체 저장"** 버튼 클릭

> **주의**: 저장 버튼을 누르기 전까지는 데이터가 저장되지 않습니다.

---

### 측정 항목별 입력 단위

| 체력 요소 | 측정 종목 | 입력 단위 |
|----------|---------|---------|
| 심폐지구력 | 왕복오래달리기 | 횟수 (회) |
| 심폐지구력 | 오래달리기-걷기 | 시간 (초) |
| 심폐지구력 | 스텝검사 | 지수 (소수점 가능) |
| 근력·근지구력 | 윗몸말아올리기 | 횟수 (회) |
| 근력·근지구력 | 악력 | 무게 (kg) |
| 유연성 | 앉아윗몸앞으로굽히기 | 길이 (cm) |
| 순발력 | 50m 달리기 | 시간 (초, 소수점) |
| 순발력 | 제자리멀리뛰기 | 거리 (cm) |
| BMI | (자동 계산) | 키·몸무게로 자동 계산 |

---

### 입력 중 자리를 비워야 할 때

인터넷이 끊기거나 급하게 자리를 비워야 할 경우:
- 입력한 내용은 **임시저장**됩니다
- 다시 접속해서 같은 학급을 열면 이전에 입력하던 내용이 그대로 있습니다
- 이어서 입력 후 저장하면 됩니다

---

### 이미 저장한 데이터 수정하기

1. 홈에서 해당 학급 카드 클릭
2. 수정할 학생의 측정값 클릭 후 값 변경
3. **"학급 전체 저장"** 다시 클릭

---

## STEP 4. 대시보드로 분석하기

> 입력한 데이터를 그래프와 통계로 확인하는 곳입니다.

---

### 대시보드 열기

1. 화면 상단 메뉴에서 **"대시보드"** 클릭

---

### 필터 사용하기

화면 상단에 필터 버튼들이 있습니다.

| 필터 | 설명 |
|------|------|
| 학년도 | 연도별로 보기 (2024, 2025, 2026 ...) |
| 학년 | 1학년만, 2학년만, 또는 전체 |
| 반 | 특정 반만 보기 |
| 성별 | 남학생만, 여학생만, 또는 전체 |

원하는 필터를 클릭하면 해당 조건의 통계가 즉시 업데이트됩니다.

---

### 대시보드 화면 설명

**홈 대시보드** (기본 화면):
- **KPI 카드**: 평균 등급, 1등급 비율, 5등급 비율 등 핵심 지표
- **등급 분포 차트**: 1~5등급 학생 수 막대 그래프
- **학년 진도**: 학급별 측정 완료 현황

**전체 분석** (상단 메뉴 → 대시보드 → 전체 분석):
- 레이더 차트 (체력 요소별 균형)
- 성별 비교 차트
- 연도별 추이 차트
- BMI 산점도

**학급별 통계** (상단 메뉴 → 대시보드 → 학급별):
- 특정 학급의 상세 통계

**학생 개별 조회** (상단 메뉴 → 대시보드 → 학생별):
- 학생 이름으로 검색
- 개인 체력 성장 추이

> **데이터가 최신이 아닌 것 같다면?**
> 데이터는 30초마다 자동으로 새로고침됩니다. 즉시 업데이트하려면 브라우저에서 **F5** 키를 누르거나 새로고침 버튼을 클릭하세요.

---

## STEP 5. 보고서 출력하기

> PDF 또는 엑셀 파일로 보고서를 출력합니다.

---

### 학급 보고서 (PDF)

1. 상단 메뉴 **"대시보드"** → **"보고서"** 탭 클릭
2. **"학급 보고서"** 섹션에서 학년·반 선택
3. **"PDF 내보내기"** 버튼 클릭
4. 브라우저가 PDF 파일을 다운로드하거나 인쇄 창을 엽니다

---

### 개인 성장 카드 (학생별 PDF)

1. **"보고서"** 탭에서 **"개인 성장 카드"** 섹션 찾기
2. 특정 학생을 선택하거나 **"전체 PDF 일괄 출력"** 버튼 클릭
3. 모든 학생의 개인 성장 카드가 하나의 PDF로 만들어집니다

---

### 원시 데이터 내보내기 (엑셀)

학교 업무 시스템에 데이터를 올려야 할 때 사용합니다.

1. **"보고서"** 탭에서 **"원시 데이터"** 섹션 찾기
2. **"엑셀 내보내기"** 버튼 클릭
3. `.xlsx` 파일이 다운로드됩니다

---

## STEP 6. 다음 해 사용하기 (연도 변경)

새 학년도가 시작되면 설정을 업데이트하세요.

1. 왼쪽 메뉴 **"설정"** 클릭
2. **학년도**를 새 연도로 변경 (예: 2026 → 2027)
3. **"저장"** 클릭
4. 새 학생 명단을 다시 등록하면 됩니다

> **이전 연도 데이터는 삭제되지 않습니다.** 대시보드에서 학년도 필터로 이전 데이터를 언제든 다시 볼 수 있습니다.

## 자주 묻는 질문 (FAQ)

---

**Q. 로그인했는데 아무 데이터도 안 보입니다**

→ 두 가지를 확인하세요.
1. 화면 상단에 학교 이름이 보이는지 확인 (온보딩 완료 여부)
2. **F5** 키로 새로고침
3. 계속되면 **"설정"** 메뉴 → **"데이터 새로고침"** 버튼 클릭

---

**Q. "연동 테스트 실패" 오류가 납니다**

→ Sheet ID 복사 방법을 다시 확인하세요.

```
https://docs.google.com/spreadsheets/d/★이 부분만 복사★/edit#gid=0
```

- `/d/` 바로 뒤부터 그 다음 `/` 전까지만 복사합니다
- 앞뒤 빈칸이 없는지 확인합니다
- Sheets 파일이 로그인한 계정 소유인지 확인합니다

---

**Q. 팝업 창이 뜨지 않습니다 (로그인이 안 됩니다)**

→ 브라우저가 팝업을 차단하고 있습니다.

1. 주소창 오른쪽 끝에 있는 차단 아이콘 클릭
2. **"paps-ims.vercel.app의 팝업 허용"** 선택
3. 페이지 새로고침 후 다시 로그인 시도

---

**Q. 측정값을 입력했는데 등급이 이상합니다**

→ PAPS 공식 기준표(교육부)를 기준으로 계산됩니다.
기준이 업데이트됐거나 학교별 기준이 다른 경우, **Google Sheets에서 직접** `grades_standard` 시트의 기준값을 수정할 수 있습니다.

---

**Q. 학생을 잘못 등록했습니다**

→ **"학생"** 메뉴에서 해당 학생 행을 클릭 → 수정 후 저장
전학 간 학생은 삭제 대신 **비활성화**를 권장합니다 (데이터 보존).

---

**Q. "Google Sheets 업데이트가 필요합니다" 배너가 화면 위에 뜹니다**

→ 앱 버전이 업데이트되어 시트 구조를 최신으로 맞춰야 합니다.
**"Sheet 자동 업데이트"** 버튼을 클릭하면 자동으로 처리됩니다. 1~2분 소요됩니다.

---

**Q. 데이터가 사라진 것 같습니다**

→ 데이터는 선생님의 Google Drive에 저장됩니다. 앱이 아닌 Google Drive에 데이터가 있습니다.
[drive.google.com](https://drive.google.com) 에 접속해서 설정 시 만든 Sheets 파일을 직접 확인해 보세요.

---

**Q. 다른 컴퓨터나 태블릿에서 사용하고 싶습니다**

→ 같은 Google 계정으로 로그인하면 다른 기기에서도 사용할 수 있습니다.
단, 새 기기에서 처음 접속하면 Sheet ID를 다시 입력해야 합니다 (1회).

---

**Q. 학교 계정으로 로그인이 안 됩니다**

→ 학교 Google Workspace 계정은 관리자 설정에 따라 외부 앱 접근이 제한될 수 있습니다.
개인 Gmail 계정으로 사용하시거나, 학교 IT 담당자에게 "외부 OAuth 앱 접근 허용" 요청을 하세요.
